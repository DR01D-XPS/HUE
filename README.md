# Neon Crystal Maze

Самостоятельная браузерная аркада в стиле лабиринтной погони: игрок собирает светящиеся кристаллы, избегает врагов, открывает портал и проходит всё более сложные уровни.

Проект не использует оригинальные ассеты, названия, логотипы или защищённые материалы Pac-Man. Вся графика рисуется через Canvas API, а звуки генерируются через Web Audio API.

## Как запустить

Откройте файл `index.html` в браузере. Сервер и установка зависимостей не нужны.

## Возможности

- управление стрелками и WASD;
- плавное движение по клеточному лабиринту;
- 5 готовых уровней;
- автоматическая генерация новых проходимых лабиринтов;
- 3 типа врагов: преследователь, случайный враг и перехватчик;
- бонусные кристаллы, которые временно делают врагов уязвимыми;
- очки, жизни, рекорд в `localStorage`;
- стартовое меню, настройки, пауза, победа уровня и Game Over;
- экранные кнопки управления для мобильных устройств;
- неоновая Canvas-графика, частицы и простые звуковые эффекты.

## Структура проекта

- `index.html` - разметка игры и экранов интерфейса;
- `style.css` - внешний вид, адаптивность и оверлеи;
- `script.js` - подключение UI, клавиатуры и мобильного управления;
- `game.js` - основной игровой цикл, рендер, столкновения и состояния;
- `player.js` - логика движения игрока;
- `enemies.js` - логика врагов и их поведения;
- `levels.js` - готовые уровни и общие константы;
- `generator.js` - генерация лабиринтов и проверка связности;
- `sound.js` - звуки на Web Audio API.

## Краткий гайд по Git

Проверить состояние файлов:

```bash
git status
```

Добавить изменения в будущий коммит:

```bash
git add .
```

Создать коммит:

```bash
git commit -m "Описание изменений"
```

Посмотреть историю:

```bash
git log --oneline
```

Подключить удалённый репозиторий GitHub:

```bash
git remote add origin git@github.com:DR01D-XPS/HUE.git
```

Отправить проект на GitHub:

```bash
git push -u origin main
```

После первой отправки следующие изменения обычно публикуются так:

```bash
git add .
git commit -m "Что изменилось"
git push
```

## SSH и GitHub

На этом компьютере уже есть SSH-ключ, и GitHub успешно принимает его для аккаунта `DR01D-XPS`.

Проверка SSH:

```bash
ssh -T git@github.com
```

Если GitHub отвечает `successfully authenticated`, ключ добавлен правильно.
