# Hue на pygame

`Hue` - небольшая 2D-головоломка с видом сверху в духе Hue. Игрок выбирает цвет мира через колесо цветов, проходит через исчезающие цветные стены, толкает ящики на нажимные плиты и открывает двери.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

Готовая Windows-сборка лежит в:

```text
dist/HUE.exe
```

## Файлы проекта

- `main.py` - основной файл игры: запускает pygame, хранит игровой цикл, уровни, управление, отрисовку, столкновения, меню и логику выбора цвета.
- `requirements.txt` - список Python-зависимостей проекта.
- `TECHNICAL_EXPLANATION.md` - подробное техническое объяснение архитектуры, механик и ключевых решений в коде.
- `.gitattributes` - настройки Git LFS; через него готовый `.exe` хранится как большой бинарный файл, а не как обычный текстовый объект git.
- `dist/HUE.exe` - готовая Windows-сборка игры, которую можно запускать без установки Python.

## Управление

- WASD или стрелки - движение по клеткам
- Удерживать SPACE - замедлить время и открыть колесо цветов
- Мышь в сторону цвета - выбрать цвет на колесе
- Отпустить SPACE - применить выбранный цвет
- R - перезапустить текущий уровень
- F11 или Alt+Enter - полноэкранный режим
- ESC - выйти в меню, закрыть колесо цвета или закрыть игру из меню
- ENTER - выбрать пункт меню или вернуться в меню после победы

## Механика

- Черные стены `#` всегда твердые и никогда не исчезают.
- Цветные стены `R`, `B`, `Y`, `V` исчезают только тогда, когда выбран такой же активный цвет.
- Если цветная стена не совпадает с активным цветом, она блокирует игрока и ящики.
- Ящики `C` можно толкать.
- Нажимные плиты `O` открывают двери `D`, когда все плиты на уровне закрыты ящиками.
- Финиш `F` переводит на следующий уровень.

В игре 6 уровней. Чем дальше, тем больше нужно сочетать выбор цвета, черные постоянные стены, ящики, плиты и двери.
