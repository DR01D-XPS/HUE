# HUE на pygame

`HUE` - 2D-головоломка с видом сверху. Игрок выбирает активный цвет через колесо цвета, проходит сквозь стены выбранного цвета, толкает ящики на плиты, открывает двери и проходит серию уровней.

## Запуск из исходников

```bash
pip install -r requirements.txt
python main.py
```

## Готовый exe

После сборки файл лежит здесь:

```text
dist/HUE.exe
```

## Управление

- WASD или стрелки - движение по клеткам.
- SPACE - удерживать колесо цвета и замедлять время.
- Мышь в сторону цвета - выбрать цвет на колесе.
- Отпустить SPACE - применить выбранный цвет.
- R - перезапустить уровень.
- M - открыть настройки громкости прямо во время игры.
- В настройках звука стрелки влево/вправо на строке `Трек` переключают музыку.
- F11 или Alt+Enter - полный экран.
- ESC - меню или назад; игру он не закрывает.
- ENTER - выбрать пункт меню.

## Звук

- В игре есть фоновая музыка, созданная процедурно через `pygame.mixer.Sound`.
- В настройках можно выбрать один из 4 треков: `HUE Theme`, `I Really Want To Stay At Your House`, `Spiral`, `Cruel Angel's Thesis`.
- Эффекты срабатывают на шаг, столкновение, смену цвета, ящик, плиту, дверь и финиш.
- В меню `Настройки звука` можно отдельно менять громкость музыки и эффектов.
- Настройки работают мышью и клавиатурой: стрелки/WASD, M для mute, ENTER/ESC для возврата.
- Громкость сохраняется между запусками в пользовательский `settings.json`.

## Механики

- Черные стены `#` всегда твердые.
- Цветные стены `R`, `B`, `Y`, `V` исчезают только при совпадении с активным цветом.
- Ящики `C` можно толкать.
- Плиты `O` открывают двери `D`, когда все плиты нажаты ящиками.
- Ловушки `X` возвращают игрока на старт уровня; ящик в ловушку толкать нельзя.
- Стрелки `^`, `v`, `<`, `>` пропускают игрока и ящики только в указанном направлении.
- Финиш `F` переводит на следующий уровень.
- После прохождения всей игры в меню появляется пункт `Выбор уровней`.
- Пасхалка: если 5 секунд упираться в стену, игрок превращается в кота.

## Структура кода

```text
main.py                         короткий запускатель
hue_game/settings.py            настройки, цвета, символы, тексты
hue_game/audio.py               музыка, звуковые эффекты, громкость
hue_game/resources.py           поиск assets в исходниках и внутри exe
hue_game/storage.py             сохранение громкости между запусками
hue_game/levels.py              карты уровней и валидация
hue_game/geometry.py            математика, easing, цвета, координаты
hue_game/model.py               логика уровня, ящики, двери, движение
hue_game/effects.py             частицы, shake, fade, floating text
hue_game/ui.py                  меню, HUD, колесо цветов, настройки, победа
hue_game/renderer.py            отрисовка игрового поля
hue_game/app.py                 игровой цикл, события, переходы
```

## Сборка exe

```bash
python -m PyInstaller --noconfirm --clean --onefile --windowed --name HUE --add-data "assets;assets" main.py
```
